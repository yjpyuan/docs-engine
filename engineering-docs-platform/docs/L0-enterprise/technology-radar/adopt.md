# 技术雷达 - 采用（Adopt）

> 经过验证的成熟技术，推荐在生产环境广泛使用

---

## 一、语言与框架

### 1.1 Java 生态

| 技术 | 版本要求 | 说明 |
|------|----------|------|
| Java | 17 LTS | 企业标准版本，长期支持 |
| Spring Boot | 3.x | 微服务开发首选框架 |
| Spring Cloud | 2023.x | 微服务治理套件 |
| MyBatis-Plus | 3.5.x | ORM 框架，简化 CRUD |

### 1.2 前端技术

| 技术 | 版本要求 | 说明 |
|------|----------|------|
| Vue.js | 3.x | 管理后台首选框架 |
| TypeScript | 5.x | 类型安全，大型项目必选 |

## 七、使用要求

```yaml
adopt_requirements:
  - 新项目必须优先选用 Adopt 列表技术
  - 使用非 Adopt 技术需技术负责人审批
  - 定期评估技术健康度（每季度）
  - 关注安全漏洞并及时升级
```
